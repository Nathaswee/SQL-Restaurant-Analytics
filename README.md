# Consumer & Restaurant Analytics — MySQL Project

## Overview

Designed and analyzed a restaurant analytics database using MySQL to uncover customer behavior, restaurant performance, and satisfaction trends.

## Key Features

* Designed normalized relational database schema
* Created primary and foreign key relationships
* Performed advanced SQL analysis using JOINs and subqueries
* Generated insights on customer ratings and restaurant performance

## SQL Concepts Used

* INNER JOIN
* LEFT JOIN
* GROUP BY
* HAVING
* Subqueries
* Aggregate Functions
* ORDER BY

## Business Insights

* Identified top-rated restaurants
* Analyzed customer satisfaction trends
* Compared restaurant performance metrics
* Extracted actionable operational insights
